# Traditional ML Projects

This folder contains classic machine learning projects with APIs, dashboards, and deployment support.

## Projects

**04_stock_forecasting**: Stock price prediction
**05_computer_vision**: Image classification and detection
**06_nlp_text_analysis**: Sentiment, NER, summarization
**07_fraud_detection**: Fraud detection in transactions
**08_customer_churn**: Churn prediction
**09_supply_chain_optimization**: Supply chain analytics
**10_energy_prediction**: Energy consumption forecasting
**11_autonomous_vehicle**: Autonomous vehicle ML

## How to Run

- Each project has its own FastAPI backend (`app.py` or `src/api/app.py`) and, where available, a Streamlit dashboard (`dashboard.py`).
- Use the one-click launcher from the repo root or `python launch_demo.py` in each project folder.

## Deployment

- Dockerfiles and cloud deployment guides included per project.

See each subfolder for detailed README and usage.

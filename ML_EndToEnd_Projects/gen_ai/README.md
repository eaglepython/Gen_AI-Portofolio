# GenAI Projects

This folder contains advanced Generative AI projects with APIs, dashboards, and deployment support.

## Projects

- **11_ai_code_generator**: Code generation from natural language prompts
- **12_ai_content_creator**: Automated text and image generation
- **13_document_intelligence**: Document parsing and analysis
- **14_conversational_ai**: Chatbots and conversational models
- **15_drug_discovery_ai**: AI for drug discovery

## How to Run

- Each project has its own FastAPI backend (`app.py`) and, where available, a Streamlit dashboard (`dashboard.py`).
- Use the one-click launcher from the repo root or `python launch_demo.py` in each project folder.

## Deployment

- Dockerfiles and cloud deployment guides included per project.

See each subfolder for detailed README and usage.
